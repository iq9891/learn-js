# 03 - 装饰图片

## 示例
![图片示例](demo.gif)

## 目录结构
``` bash
├── demo.gif                     # 示例图片
├── index.html                   # 排好版的页面
├── script.js                    # 写 JavaScript 的地方
└── style.css                    # 写 CSS 的地方
```

## 功能
1. 改变内边距
2. 改变模糊程度
3. 改变基础颜色

## 时间
1. 第一版建议用时 2  小时
2. 优化版建议用时 30 分钟
