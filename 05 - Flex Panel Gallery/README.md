# 04 - 窗帘效果

## 示例
![图片示例](demo.gif)

## 目录结构
``` bash
├── demo.gif                     # 示例图片
├── index.html                   # 排好版的页面
└── script.js                    # 写 JavaScript 的地方
```

## 功能
1. 点击细图片，图片撑大
2. 图片撑大之后文字动态入场
3. 点击宽图，图片还原
4. 图片还原之后文字动态出场

## 时间
1. 第一版建议用时 2  小时
2. 优化版建议用时 30 分钟
